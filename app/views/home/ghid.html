1. Introduction

1.1 General description

Terrorism Web Visualiser(TeW) is a tool designed to allow flexible visualisation of global terrorism related data. It contains approximately 180000 attacks from 1970 to 2017 around the world, a lot of details about the attacks and a short summary of the event.

2. User

2.1 Statistics

Users can search for attacks using filters and the searchbar. Users can see the date and a short summary of the attack on the page, but they can press the button "See More" to visualise every aspect of that attack in the database. The selected attacks will be uset to create a chart (Pie, Bar or Line). The chart can be exported as CSV, SVG or WebP.

2.2 Map

Users can search for attacks using filters and the searchbar. The selected attacks will displayed on the map. The map can be zoomed in and out. The map can be exported as CSV, SVG or WebP.

3. Administrator

3.1 Login

Admins must introduce valid credentials in order to be redirected to administration page.

3.2 Administration

Admins can search attacks using filters in order to make searching easier. After finding an attack they can edit, delete or visualise it using the buttons at the button of the article.

3.3 Add article

Adding an article is done by pressing the "Add Article" button on the top of administration page. Admins will be redirected on another page. On the page it will be displayed inputs where the admin can insert data and the corresponding column.

3.4 Edit

Editing an attack is done by pressing the "Edit" button on the article. Admins will be redirected on another page. On the page it will be displayed the actual data about the attack, the corresponding column, and an input where the admin can insert the new data.

3.5 See more

Visualising an attack is done by pressing the "See More" button on the article. Admins will be redirected on another page. On the page it will be displayed data about the attack.

3.6 Delete

Deleting an attack is done by pressing the "Delete" button on the article. Admins will be redirected on another page. On the page it will be displayed a message of whether or not the attack was succesfully deleted or not.
